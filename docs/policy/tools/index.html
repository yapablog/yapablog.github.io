はじめに

本ブログでは、論文解説記事に加えて、
筆者が独自に開発した JavaScript ベースの技術ツールを公開しています。

本ページでは、これらツールについて、

  • 著作権の所在
  • 利用・改変の可否
  • 再配布や商用利用の扱い
  • データ・プライバシーに関する考え方

を明確にします。

著作権の所在について

本ブログ上で公開している JavaScript ツールのソースコードおよびロジックは、
特に明示がない限り、すべて本ブログ運営者（筆者）に帰属します。

ただし、これは「利用を制限する」ためではなく、
利用条件を明確にするための形式的な記述です。

利用条件（ライセンスの考え方）

基本方針

本ブログで公開している JavaScript ツールは、原則として自由に利用可能です。

以下の行為をすべて許可します。

  • 個人利用・研究利用・教育利用
  • 商用・非商用を問わない利用
  • ソースコードの改変
  • 自作ツールや別プロジェクトへの組み込み
  • 再配布（そのまま／改変後を問わない）

いわゆる MIT License に近い考え方を採っています。

クレジット表記について

クレジット表記は必須ではありませんが、
もし言及いただける場合は以下のような形を推奨します。

Based on a web-based tool originally published at https://yapablog.github.io/

これは義務ではなく、あくまで任意です。

保証および責任の範囲

これらのツールは、

  • 教育目的
  • 概念理解
  • 初期設計検討
  • 思考実験・可視化

を主目的として提供されています。

そのため、

  • 計算結果の正確性
  • 特定用途への適合性
  • 実運用・実設計への適用

について、いかなる保証も行いません。

本ツールの利用によって生じたいかなる損害についても、
筆者は責任を負いません。

データ取得・プライバシーについて

本ブログは 静的サイト として運営されています。

入力データの扱い

  • ツールに入力された数値やパラメータは
    すべて閲覧者のブラウザ内でのみ処理されます。
  • サーバー側に送信・保存されることはありません。
  • そもそも、技術的に取得する仕組みを持っていません。

したがって、

  • 入力内容が記録されること
  • 個人情報が収集されること

はありません。

クッキー・トラッキングについて

  • 独自のクッキーやトラッキングスクリプトは使用していません。
  • アクセス解析を行う場合でも、一般的な静的解析に限定されます。

外部ライブラリについて

ツール内で外部ライブラリ（例：数値計算・描画ライブラリ）を使用している場合があります。

それらについては、

  • 各ライブラリのライセンスに従って利用しています
  • ライブラリ自体の著作権は、各開発者に帰属します

詳細は、各ツールのソースコード内コメントまたは公式サイトをご確認ください。

方針の変更について

本ページの内容は、
ツールの拡充やブログ運営方針の変更に伴い、予告なく更新されることがあります。

ただし、既に公開されているツールの利用を遡って制限することはありません。

まとめ

本ブログの JavaScript ツールについての考え方を、一言でまとめると以下の通りです。

自由に使って、自由に改変して構いません。
入力データは取得しておらず、こちらに届くこともありません。
ただし、結果の利用は自己責任でお願いします。

安心して、学習・研究・開発に役立てていただければ幸いです。
